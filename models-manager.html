💡 How It Works
OpenRouter is a unified API that lets you access 200+ LLMs from one endpoint. By pairing it with Claude Code, you can swap models on the fly.

Manual usage:
claude --model openrouter/model-id

Aliases: create .bat / .sh shortcut files so you can simply type claude-gpt4 or claude-gemini in your terminal.

Pricing: OpenRouter shows per-token pricing. "Prompt" = input tokens, "Completion" = output tokens. Models marked FREE have no per-token cost.
