Welcome to Claude Code Dashboard

This dashboard helps you manage every aspect of Claude Code from your browser — servers, tools, conversations, costs, permissions, and more.

🚀 Quick Start Guide

  1. Start the server
    node mcp-manager-server.js
  2. Open the dashboard

    Navigate to http://localhost:3456 in your browser.

  3. Explore the navigation

    Use the navigation bar at the top to browse all available pages.

  4. Check your setup

    Start with the Health page to verify everything is working correctly.

  5. Manage your MCP servers

    Go to Servers to see, add, or configure your MCP server connections.

🔌 What is MCP?

Model Context Protocol (MCP) is an open standard that connects Claude to external tools and data sources. Think of it as a universal adapter between AI and the services you already use.

GitHub, Slack, DBs...
  • Servers provide tools — each MCP server exposes capabilities like reading files, creating PRs, querying databases, or sending messages.
  • Claude discovers tools automatically — once a server is connected, Claude sees its tools and can use them without extra configuration.
  • Example: Add a GitHub MCP server, and Claude can now create pull requests, read issues, review code, and manage repositories.

💻 What is Claude Code?

Claude Code is an AI-powered command-line tool for software development. It runs directly in your terminal and can:

  • Read and write files in your project
  • Run shell commands like build, test, and deploy
  • Search and navigate large codebases intelligently
  • Connect to MCP servers for extended capabilities (GitHub, databases, APIs, and more)
  • Remember context across conversations with memory and session resumption

This dashboard gives you a visual interface to manage Claude Code's configuration, review your usage, and explore its full feature set.

🔄 How to Resume a Conversation

  1. Go to the Sessions or Conversations page from the navigation.
  2. Find your chat by topic, date, or keyword search.
  3. Click "Resume" — this copies the resume command to your clipboard.
  4. Paste the command in your terminal:
    claude --resume SESSION_ID
  5. Or, to quickly continue the most recent conversation:
    claude --continue

🔧 Common Issues & Fixes

"MCP server not connecting"
Check the Health page for diagnostics. Run Config Doctor (also on the Health page) to validate your configuration files and find mismatches.
"Too many tokens used"
Visit the Tools page to see per-tool token impact. Disable or remove tools you do not use to reduce token consumption.
"Permission prompts are annoying"
Go to the Permissions page and set up permission profiles. You can pre-approve trusted tools and commands to skip confirmation prompts.
"Can't find an old conversation"
Use the search feature on the Conversations page. You can filter by date, keywords, or session metadata.
